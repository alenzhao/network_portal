About the Network Portal

version: {{version}}

The Network Portal is a database of gene transcription regulatory networks, enabling exploration, annotation and comparative analysis...

Credits

The Network Portal is developed in the Baliga Lab at the Institute for Systems Biology as part of ENIGMA.

Tools

Python, Django, NetworkX, Solr, PostgeSQL, Cytoscape Web, jQuery, the R project

Data sources include: Microbes Online, KEGG, NCBI, GO, TIGR

Code

The source code for the Network Portal is available on GitHub.
